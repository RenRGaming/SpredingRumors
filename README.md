# Интерактивная модель распространения слухов

Данный проект представляет собой стохастическую агентную модель для исследования динамики распространения информации в различных сетевых структурах. Включает интерактивную визуализацию процесса в реальном времени.

## Описание модели
В системе каждый агент может находиться в одном из трех состояний:
* **Unaware (Зеленый)** — агент не осведомлен о слухе.
* **Spreader (Красный)** — агент активно распространяет слух среди соседей.
* **Silent (Желтый)** — агент знает слух, но не передает его.

Процесс перехода между состояниями стохастичен и зависит от параметров «интересности» ($\beta$), «актуальности» ($T$) и индивидуальной «болтливости» агентов ($\alpha$).

3. Для настройки параметров используйте появившиеся интерактивные виджеты.

## Функционал визуализации
В репозитории реализованы два основных типа симуляций:

### 1. Моделирование на регулярных сетках
Позволяет исследовать распространение на 2D-решетках с выбором типа соседства:
* **Окрестность фон Неймана** (взаимодействие с 4 ближайшими соседями).
* **Окрестность Мура** (взаимодействие с 8 ближайшими соседями).

### 2. Моделирование на кластерных графах
Использует модель стохастических блоков (Stochastic Block Model) для имитации социальных сообществ. Доступна настройка числа кластеров, а также плотности внутренних ($p_{in}$) и внешних ($p_{out}$) связей.
